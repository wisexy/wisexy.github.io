WisexyJavaScript学习笔记：类型、值、变量 2017-03-13

JavaScript中的类型

  1. 数字
  2. 字符串（char是长为1的字符串）
  3. 布尔值
  4. null
  5. undefined
  6. 对象（包括函数）

算术运算中的注意点

  • 1/0 = Infinity（除0不报错）
  • 0/0 = NaN
  • NaN不与任何值相等，包括自身
  • Infinity/Infinity = NaN
  • 负数开平方 = NaN
  • +0 == -0 为真
  • 判断 NaN 和 Infinity 的好办法是 isNaN() 和 isFinite()

字符集

  1. JavaScript采用UTF-16编码的Unicode字符集。
  2. 不能表示为16位的Unicode字符用两个16位值组成的序列表示，称作“代理项对”。
  3. JavaScript定义的各种字符串操作方法均作用于16位值，而非字符，且不会对代理项做单独处理。

转义字符

  • \xXX 由两位十六进制数XX制定的Latin-1字符
  • \uXXXX 由4位十六进制数XXXX指定的Unicode字符

null 和 undefined

  • null表示”没有对象”，即该处不应该有值。典型用法是：
  1. 作为函数的参数，表示该函数的参数不是对象。
  2. 作为对象原型链的终点。
  • undefined表示”缺少值”，就是此处应该有一个值，但是还没有定义。典型用法是：
  1. 变量被声明了，但没有赋值时，就等于undefined。
  2. 调用函数时，应该提供的参数没有提供，该参数等于undefined。
  3. 对象没有赋值的属性，该属性的值为undefined。
  4. 函数没有返回值时，默认返回undefined。
var i;
i // undefined

function f(x){console.log(x)}
f() // undefined

var  o = new Object();
o.p // undefined

var x = f();
x // undefined
